# Swift Composites

---

```swift
// • –––––––––– • –––––––––– • –––––––––– •
// | Array      | Dictionary | Enum       |
// • –––––––––– • –––––––––– • –––––––––– •
// | Set        | Tuple      |            |
// • –––––––––– • –––––––––– • –––––––––– •


Homogenous
var a_: [Int] = [1, 2, 3]
var s_: Set<Int> = [1, 2, 3]


Heterogeneous
var d_: [Character: Int] = ["N": 47, "W": 122]
var t_: (String, Int, Bool) = ("SEA", 206, true)
```
